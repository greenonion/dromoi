# Dromoi

This site helps people explore Greek folk music scales built from tetrachords and pentachords using 12-TET notes. It renders scales on a staff and highlights them on a keyboard for quick listening and study.

## Tech
- Plain HTML, CSS, and JavaScript
- VexFlow for staff rendering
- Bun for dev server, bundling, and tests

## Commands
- `bun run dev` to build (watch) and serve the site locally
- `bun run build` to bundle into `dist/`
- `bun test` to run logic tests
